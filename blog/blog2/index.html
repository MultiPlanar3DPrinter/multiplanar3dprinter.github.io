2022-02-05 (Updated 2022-03-08)

A general summary of project progress as of February 5th, 2022.

Mechanical

Built and assembled a completely 3D printed version of the bracket and hot end gears to see if the geometry was feasible. Followed this up by using learnings from that prototype to build a version using aluminum shafts for better movement and stiffness. Learned several things as a result of these prototypes:

  • Geometric tolerances for 3D printing fits
  • Shaft sizing
  • Bending radius for the bowden tube
